## Linting and Testing

From the repo root run:

1. This command to run mypy accorss sources and tests
    ```bash
    mypy src
    ```
2. This command to run tests:
    ```bash
    pytest
    ```
3. This command to run flake8 linter:
    ```bash
    flake8 src tests
    ```
4. This command to sort imports
    ```bash
    isort src tests
    ```
5. This command to run tox:
    ```bash
    tox
    ```
